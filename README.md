# Blarg-in-Torn
Blargbot code for various API & Non API ccmds for Torn City.

Current Commands

Entrance:
greet,
verify,
setapikey


API Needed:
profile


Non Api Needed:
vitals, 
request,
withdraw


If the command DOES NOT start with "..cc set" it needs to be added through Blargbot's IDE, otherwise it can be fired off in the server.
